# Nazim Uddin Portfolio Website

This is a static HTML, CSS, and JavaScript portfolio website ready for GitHub Pages.

## Files

- `index.html` — main web page
- `style.css` — responsive styling and dark mode
- `script.js` — project cards, mobile menu, active nav, theme toggle, animations
- `images/profile.jpg` — profile photo
- `images/` — place your project images here
- `assets/Nazim_Uddin_CV.pdf` — downloadable CV

## Project image names

The project cards currently expect these image filenames:

- `images/project-oil-spill.jpg`
- `images/project-wetland.jpg`
- `images/project-flood.jpg`
- `images/project-landuse.jpg`
- `images/project-hydro.jpg`
- `images/project-agriculture.jpg`

You can replace these images, or edit the `projects` array in `script.js` to use different filenames.

## GitHub Pages setup

1. Create a GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save and wait for GitHub to publish the site.
